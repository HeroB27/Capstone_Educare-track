### 1. Parent Module - Receiving Wrong Alerts
**Problem:** Parents were receiving announcements or alerts for students not linked to them, causing potential gossip issues.

**Root Cause:** The notification filtering was not robust enough - it only checked message content for student names but didn't properly verify parent-child linkage.

**Solution:** Enhanced `loadNotifications()` in parent-notifications.js with more robust filtering:
- Check both message and title for child names
- Use Set for efficient ID matching
- Include announcements (general)
- Properly exclude unrelated notifications

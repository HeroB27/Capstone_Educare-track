# Plan: Create Shared Helper File - attendance-helpers.js

## Overview
Create a shared helper file `core/attendance-helpers.js` containing three attendance calculation functions:
1. getTotalSchoolDays(startDate, endDate, gradeLevel)
2. countUnexcusedAbsentDays(studentId, startDate, endDate)
3. getAttendanceRate(studentId, startDate, endDate)

## Requirements
- Exclude Sundays from school day count
- Exclude full-day holidays from school day count
- Exclude suspensions from school day count
- Partial holidays still count as school days
- Approved excuse letters reduce absence counts
- Half-day absences count as 0.5

## Implementation Details

### Function 1: getTotalSchoolDays(startDate, endDate, gradeLevel)
**Purpose**: Calculate total number of school days between two dates for a specific grade level
**Parameters**:
- startDate: String in YYYY-MM-DD format
- endDate: String in YYYY-MM-DD format
- gradeLevel: String (optional) - e.g., "Grade 1", "Kinder"

**Steps**:
1. Fetch holidays from `holidays` table within date range
2. Filter for full-day holidays only (time_coverage = 'Full Day')
3. Apply grade level filtering if provided (target_grades contains 'All' or specific grade)
4. Fetch active suspensions within date range
5. Build set of all dates covered by suspensions
6. Iterate through each date from startDate to endDate:
   - Skip if Sunday (dayOfWeek = 0)
   - Skip if date is in fullDayHolidays set
   - Skip if date is in suspendedDates set
   - Otherwise increment counter
7. Return final count

### Function 2: countUnexcusedAbsentDays(studentId, startDate, endDate)
**Purpose**: Count unexcused absent days for a student in date range
**Parameters**:
- studentId: Integer - student ID
- startDate: String in YYYY-MM-DD format
- endDate: String in YYYY-MM-DD format

**Steps**:
1. Fetch attendance_daily_summary records for student in date range
2. Fetch approved excuse letters for student in date range (status = 'Approved')
3. Create map of date -> absence_type from excuses
4. For each summary record:
   - Skip if fully excused (excuse = 'whole_day')
   - Check morning and afternoon status
   - Apply partial day excuses:
     * Skip morning absence if excuse = 'half_day_morning'
     * Skip afternoon absence if excuse = 'half_day_afternoon'
   - Calculate unexcused portions:
     * Both periods absent = 1.0 day
     * One period absent = 0.5 day
5. Return total unexcused days

### Function 3: getAttendanceRate(studentId, startDate, endDate)
**Purpose**: Calculate attendance rate percentage for student over period
**Parameters**:
- studentId: Integer - student ID
- startDate: String in YYYY-MM-DD format
- endDate: String in YYYY-MM-DD format

**Steps**:
1. Fetch attendance_daily_summary records for student in date range
2. If no records, return 0
3. Initialize counters:
   - presentPeriods = 0
   - totalPeriods = 0
4. For each summary record:
   - Morning period:
     * Increment totalPeriods
     * If status in ['Present', 'Late', 'Excused'], increment presentPeriods
   - Afternoon period (if not null):
     * Increment totalPeriods
     * If status in ['Present', 'Late', 'Excused'], increment presentPeriods
5. Return (presentPeriods / totalPeriods) * 100

## Database Dependencies
- Table: holidays (holiday_date, time_coverage, target_grades, is_suspended)
- Table: suspensions (start_date, end_date, is_active)
- Table: attendance_daily_summary (student_id, date, morning_status, afternoon_status)
- Table: excuse_letters (student_id, date_absent, absence_type, status)

3. Half-day inference – from subject attendance
4. Admin attendance summary – with half-day counts

## Key Implementation Details

### Half-Day Inference Logic
1. Get all subject loads for student's class
2. Get attendance logs for all subjects on that date
3. Classify subjects by time (morning: start < 12:00, afternoon: start >= 12:00)
4. If any morning subject is Absent → morning half = Absent
5. If any afternoon subject is Absent → afternoon half = Absent
6. Upsert result to `attendance_daily_summary`

### Global Function Exposure
Ensure `recomputeHalfDayBatch` is available globally:
```javascript
window.recomputeHalfDayBatch = recomputeHalfDayBatch;
```

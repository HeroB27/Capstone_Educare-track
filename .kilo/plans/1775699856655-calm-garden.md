# Plan: Class Attendance Table in Data Analytics

## Objective
Add a new "Class Attendance Overview" table to the admin Data Analytics page that shows per-class attendance breakdown, with drill-down to individual student attendance on row click.

## Current State
- **admin-class-management.js** (lines 620-832): Has existing attendance per class view that calculates present, late, absent, excused counts per student within a class
- **admin-data-analytics.js** (lines 307-370): Has `fetchClassPerformance()` that calculates attendance rate per class, but only returns a single rate number (not the breakdown)

## Changes Required

### 1. admin-data-analytics.js
- Create new function `fetchClassAttendanceBreakdown(dateStart, dateEnd)` that calculates for each class:
  - Total attendance records
  - Present count
  - Late count  
  - Absent count (unexcused)
  - Excused count
  - Half-day count
- Calculate percentages for each metric
- Update the analytics loading flow to include this new data

### 2. admin-data-analytics.html
- Add new section after "Class Performance" card with:
  - A table showing: Class Name | Present % | Late % | Absent % (red bg) | Excused % (blue bg) | Half Day %
  - Clickable rows that open a modal/section showing individual student attendance for that class

### 3. Modal/Drill-down View
- On row click, show modal with:
  - Date filter (inherits from main analytics dates)
  - Student list with individual attendance breakdown
  - Same columns as class-level but per student

## Column Logic
- **Absent % (red)**: Unexcused absences only
- **Excused % (blue)**: Excused absences from approved excuse letters
- **Half Day %**: Half-day absences (morning OR afternoon only)
- **Late %**: Late arrivals
- **Present %**: On Time status

## Notes
- Uses existing date range selector on page
- Excused detection uses `excuse_letters` table with status 'Approved'
- Half-day detection: `morning_absent !== afternoon_absent` (one true, one false)

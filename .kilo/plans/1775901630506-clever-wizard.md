# Plan: Add Start and End of School Days to Attendance Settings

## Goal
Add "School Start Time" and "School End Time" settings in the admin Attendance Settings page, and have other modules (guard, teacher) use these as fallback defaults.

## Current State
- Admin attendance settings saves to `settings` table (key-value pairs)
- `grade_schedules` table has per-grade start/end times
- Guard module uses grade_schedules with hardcoded fallback: `07:30:00` to `15:00:00`

## Implementation Steps

### 1. Add UI fields in admin-attendance-settings.html
- Extend "Time Thresholds" tab
- Add inputs for:
  - `school_start_time` - Start of School Day
  - `school_end_time` - End of School Day

### 2. Update admin-attendance-settings.js
- Add keys to `saveThresholds()`: `['school_start_time', 'school_end_time']`

### 3. Update guard-core.js (lines 89-95)
- Query `settings` table for `school_start_time` / `school_end_time`
- Use as fallback when grade_schedules is not found
- Priority: grade_schedules → settings table → hardcoded default

## Settings Keys to Add
| Key | Description | Default |
|-----|--------------|---------|
| `school_start_time` | Global school day start | `07:30:00` |
| `school_end_time` | Global school day end | `15:00:00` |

1. `admin/admin-attendance-settings.html`
2. `admin/admin-attendance-settings.js`

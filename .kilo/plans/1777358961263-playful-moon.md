   - Impact: Teacher cannot see homeroom attendance
   - Mitigation: Add summary-based fallback path

   - Impact: Teacher analytics incomplete
   - Mitigation: Use summary with student_id filter

3. **`teacher/teacher-data-analytics.js:populateStudentAttendanceTable()`** (modal)
   - Risk: Individual student view breaks
   - Impact: Low - could use logs (acceptable for detail view)

   - Risk: Same as above
   - Impact: Dead code

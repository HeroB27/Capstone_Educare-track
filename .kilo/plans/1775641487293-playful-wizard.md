## 2. Half-Day Attendance Detection

### Uses Existing Fields
- `morning_absent`: true if student absent in AM
- `afternoon_absent`: true if student absent in PM

### Logic for Analytics
- **Full-day absence**: Both AM and PM absent (or whole-day status)
- **Half-day absence**: Only AM absent OR only PM absent
- **Present**: Either AM or PM present counts as half-day present

### YTD Half-Day Calculation
Query: `COUNT(*) WHERE (morning_absent = true OR afternoon_absent = true)` per student
